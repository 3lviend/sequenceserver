SequenceServer

Easily BLAST & share your sequence data.

Features

Easy setup:

  • No configuration necessary: uses ruby's internal web server.
  • Finds all formatted blast databases.

Easy to use:

  • Elegant & helpful interface.
  • Deactivates inappropriate blast methods depending on type of entered sequence.
  • Deactivates inappropriate databases based on selected blast method.
  • Single-click retrieval of hit sequences directly from BLAST results.
  • "The iPod of Blast searching"

Requirements

  • ruby 1.8.7 with rubygems (if you have git, get ruby with RVM).
  • NCBI Blast+ 2.2.24
  • Sequence databases formatted for BLAST (with formatdb or makeblastdb; see FAQ)

Download & Setup

Will be easier soon. For now:
  • Install Sinatra: 
    gem install sinatra
  • Download sequenceserver. 
    wget http://github.com/yannickwurm/sequenceserver/zipball/master
  • Edit the config.yml file to indicate the location of your blast databases.
  • Launch the server: 
    ruby search.rb
  • Access SequenceServer on http://localhost:4567
