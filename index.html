SequenceServer Beta

Easily set up a local BLAST web server to search & share sequence data.

We are still working on improving Sequenceserver to make it become "The iPod of Blast searching". But it already provides an easily useable BLAST server (see demo and setup below). Send us a quick note and we'll keep you updated on releases.

Features

Easy to setup:

  • Helps you batch format databases.
  • Finds all formatted blast databases.
  • No configuration necessary: uses ruby's internal web servers. Or use Apache with Rake.

Easy to use:

  • Elegant & helpful interface.
  • Deactivates inappropriate blast methods depending on type of entered sequence.
  • Deactivates inappropriate databases based on selected blast method.
  • Single-click retrieval of hit sequences directly from BLAST results (you can easily add custom links)

Demo server

Ant genome sequence blast search: running a pre-release version of SequenceServer. Hosted by the ant genome database.

Requirements

  • ruby 1.8.7 or later (if you have git, get ruby with RVM).
  • NCBI Blast+ 2.2.24
  • FASTA sequence databases formatted for BLAST (with formatdb or makeblastdb; see FAQ)

Download & Setup

More details in the readme file. Basically:
  • Download sequenceserver. 
    wget http://github.com/yannickwurm/sequenceserver/zipball/master
  • Unzip, and move to Sequence Server's directory, and do: 
    gem install bundler && bundle install
  • Edit the example.config.yml file to indicate the location of your blast databases. Or create some with databaseformatter.rb
  • Launch the server: 
    ruby sequenceserver.rb
  • Access SequenceServer on http://localhost:4567
